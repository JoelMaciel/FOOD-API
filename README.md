# Tecnologias usadas.

- Java
- Spring Boot
- AWS , Docker , Maven
- Spring Security, OAuth2 , JWT
- Spring Data JPA , Spring Hateoas
- Swagger , OpenAPI
- MySQL

## Deploy em produção

- Back end: AWS
- Database: MySQL

# Como rodar o projeto

Pré-requisito : Java 11

```bash
# clonar repositório
git clone https://github.com/JoelMaciel/FOOD-API.git

# entre na pasta do projeto
cd food_api

# executar o projeto
./mvnw spring-boot:run
```
